myFlix


Project

myFlix is a website that displays information about movies; eg.directors, movie synopsis, and genre. As a user youâ€™re able to filter movies by title using the search bar and look at information specific to the movie, director, or genre.

Creating an account allows the user to save their favorite movies to a list that can be viewed in the profile section. In the profile section the user can view and update any of their information or delete their account and all data associated with it. The user can also edit their favorite movies list displayed under their account information.

    • My Role

    Completed as part of the CareerFoundry Full-stack web development course, I was responsible for creating the back-end/server-side of the app. This is where all the information will be stored and handled.

    I was also responsible for building the front-end/client-side. This is where the users will be actively viewing and interacting with the information. The styling and theme of the app were responsibility as well.

    Project Difficulties

    This project really forced me to get creative and look for different solutions that wouldn't affect the rest of the project and hold up long term. The biggest issue was getting the project to successfully upload without crashing after continued use. I reached out to my superiors and looked at my classmates work for reference, but unfortunately didn't get very far.

    My Solution

    After some wasted time trying out different solutions and reaching out for help, I called it quits and restarted. I transferred my code over to a new project and reinstalled all packages. The code then successfully uploaded and didn't crash after prolong use. The feeling was euphoric.
