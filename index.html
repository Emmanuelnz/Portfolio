myFlix-React

myFlix is a 2-part project built as part of the Career Foundry full stack web dev course. Once registered, the app shows the user a movies basic details. It also allows the user to add and remove movies to a favorite movies list. The user can also update their profile information and delete their account. In the first part I created a REST API for the app to pull its data from. I worked with Node.js, MongoDB shell, Mongo, git, PostgreSQL, and Postman for the first time to create the database. In the second part of the project I built the front-end using React.js and React bootstrap, finally connecting both parts to create a fully functioning app. I struggled the most with this app, but it did familiarize me more with reading & debugging code. I also got to spend more time React.

M E E T

M E E T is a Progressive Web App that shows developers upcoming events around them. This project is part of the Career Foundry full stack web dev course. I used a mix of Test Driven Development and Behavior Driven Development when developing the app. During the course of the project I was introduced to AWS Lambda, Serverless, Gherkin, and working with Googles APIs. I used tools like Cucumber to write test for features, along with Jest to test those features during development.

A B O U T

My name is Emmanuel. I'm 19 years old and I really enjoy seeing things come together. I'm a minimilist and believe that less is more. I prefer things to be simple and efficient as opposed to things being over the top/cluttered and messy. These ideas can be seen in my code as well. I also really like my free time and being outside. I was looking for a job that would grant me some freedom and allow me to get creative. Although, honestly speaking, my primary concern isn't my pay but my time and freedom. I really don't like having to work weekends and holidays to meet an absurd deadline or goal. After some trail and error I found coding. I really enjoyed seeing and being involved in an apps/sites development process. I'm aiming to work hard enough to achieve a skill level that offers a decent position in the field where my career expectations can be met.
